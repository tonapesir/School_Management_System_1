📌 Template कसे बनवायचे?
  1. Word मध्ये template उघडा
  2. जिथे data यायला हवे तिथे {full_name} असे लिहा
  3. File → Save As → .docx format मध्ये save करा
  4. वरती Template Upload मध्ये upload करा
  5. 📝 Word बटण दाबल्यावर filled .docx download होईल
  6. Word मध्ये उघडून File → Export → PDF करा
